Executive Summary
StockWatch API is a containerized inventory management REST API built with FastAPI. The application allows users to create products, adjust inventory quantities, and identify low-stock items.
The solution consists of four Docker services:
•	FastAPI application 
•	PostgreSQL database 
•	Redis cache 
•	Nginx reverse proxy 
The application is designed for containerized deployment using Docker Compose and includes Continuous Integration through GitHub Actions.
No Infrastructure as Code (Terraform/CloudFormation) or deployment pipeline was identified during discovery.
